slice(start, end) : start ~ end-1 까지 요소 추출 새로운 배열 생성
a = arr.slice(1); //1 ~끝 까지

배열복사 arr.slice()
깊은복사 : 배열요소복사 - 한쪽이 변경되어도 다른쪽에 영향을 주지 않는다.
얕은복사 : 주소복사 - 한쪽이 변경되면 다른쪽에 영향을 준다.



splice(index, n,a,b,c....)index위치에서 n개를 삭제하고 a,b,c요소를 삽입
splice(index, 0, a,b,c....) n=0일때는 삭제 하지 않는다 a,b,c요소를 삽입
splice(index, n) : index위치에서 n개를 삭제
