
✅ transform 속성이란?
transform은 HTML 요소의 모양이나 위치를 변형할 때 사용하는 CSS 속성입니다.

✨ 대표적으로 가능한 변형 종류:
함수명	설명
translate(x, y)	 위치 이동 (좌우/상하)
scale(x, y)	    크기 조절 (확대/축소)
rotate(deg)	   회전
skew(x, y)	  기울이기
matrix(...)	  2D 변환을 모두 포함한 복합 형태

🔍 transform 예제들
📌 1. 위치 이동

.box {
  transform: translate(50px, 20px); /* 오른쪽 50px, 아래 20px */
}
📌 2. 회전

.box {
  transform: rotate(45deg); /* 시계방향 45도 회전 */
}
📌 3. 확대/축소

.box {
  transform: scale(1.5); /* 1.5배 확대 */
}
📌 4. 여러 변형 조합

.box {
  transform: translateX(100px) rotate(30deg) scale(1.2);
}

✅ transform의 특징
즉시 적용되며, transition과 함께 쓰면 부드럽게 변형됨.

레이아웃에는 영향을 주지 않음

즉, 요소는 보이는 위치가 바뀌지만 주변 요소의 배치는 변하지 않음.

GPU를 이용해 렌더링되기 때문에 성능이 좋음

✅ transform + transition 예제

.box {
  transition: transform 0.3s ease;
}
.box:hover {
  transform: scale(1.1) rotate(5deg);
}
→ 마우스를 올리면 자연스럽게 확대 + 회전

✅ 예시 비교
① transform + transition (간단한 변화)

.box {
  transition: transform 0.3s ease;
}
.box:hover {
  transform: scale(1.2);
}
→ 사용자 동작(hover)에 따라 변화


🎯 마무리 요약
transform: 요소를 회전, 이동, 크기 조절 등으로 변형함

transition: 변형을 부드럽게 만들어 줌
